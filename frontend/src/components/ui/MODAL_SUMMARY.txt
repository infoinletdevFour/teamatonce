================================================================================
  COMPONENTS INCLUDED
================================================================================

1. Modal (Base Component)
   - Generic modal for custom content
   - Configurable sizes: sm, md, lg, xl
   - Optional title, description, close button
   - Backdrop blur effect
   - Smooth animations

2. ConfirmModal
   - Confirmation dialogs (Yes/No, Confirm/Cancel)
   - 4 button variants: primary, danger, warning, success
   - Custom icons support
   - Loading states
   - Async action support

3. AlertModal
   - Alert messages with OK button
   - 4 types: info, success, warning, error
   - Auto-generated icons
   - Type-based styling
   - Custom icons support

4. FormModal
   - Wrapper for forms in modals
   - Built-in submit/cancel buttons
   - Form validation support
   - Loading states
   - Prevent close on validation error

5. ModalProvider
   - Context provider for global modals
   - Centralized modal management
   - Single container for all context modals

6. useModal() Hook
   - Access confirm() and alert() anywhere
   - Recommended approach for multiple modals
   - Clean, reusable pattern

7. Standalone Functions
   - confirm(message, title?, options?) => Promise<boolean>
   - alert(message, title?, options?) => Promise<void>
   - No context required
   - Perfect for one-off confirmations

  USAGE PATTERNS

Pattern 1: Context Hook (Recommended)
--------------------------------------
// Setup once in App.tsx
<ModalProvider>
  <App />
</ModalProvider>

// Use anywhere
const { confirm, alert } = useModal();
const ok = await confirm('Delete?');


Pattern 2: Standalone Functions
-------------------------------
import { confirm, alert } from '@/components/ui/Modal';
const ok = await confirm('Sure?');


Pattern 3: Component-Based
--------------------------
const [open, setOpen] = useState(false);
<ConfirmModal isOpen={open} onClose={() => setOpen(false)} ... />

================================================================================
  DESIGN SPECIFICATIONS
================================================================================

Modal Sizes:
  - sm: max-w-md (448px)    - Confirmations, alerts
  - md: max-w-lg (512px)    - Forms, default
  - lg: max-w-2xl (672px)   - Complex forms
  - xl: max-w-4xl (896px)   - Full content

Button Variants:
  - primary: Blue → Purple → Pink (default actions)
  - danger: Red gradient (delete, remove, destructive)
  - warning: Amber → Orange (caution, be careful)
  - success: Emerald → Green (confirm, proceed)

Alert Types:
  - info: Blue icon, blue background
  - success: Green icon, green background
  - warning: Amber icon, amber background
  - error: Red icon, red background

Styling:
  - Overlay: bg-black/50 backdrop-blur-sm
  - Modal: bg-white/95 backdrop-blur-xl
  - Border: border border-gray-200
  - Padding: p-6 md:p-8
  - Border Radius: rounded-2xl (16px)
  - Shadow: shadow-2xl

================================================================================
  EXAMPLE CODE
================================================================================

Delete Confirmation:
-------------------
const { confirm, alert } = useModal();

const handleDelete = async () => {
  const ok = await confirm(
    'This will permanently delete all data.',
    'Delete Project?',
    { confirmVariant: 'danger' }
  );
  
  if (ok) {
    await deleteAPI();
    await alert('Deleted!', 'Success', { type: 'success' });
  }
};


Form Modal:
----------
<FormModal
  isOpen={open}
  onClose={() => setOpen(false)}
  title="Create Project"
  onSubmit={async () => {
    await createProject(formData);
  }}
>
  <input type="text" placeholder="Project name" />
  <textarea placeholder="Description" />
</FormModal>


Chained Modals:
--------------
const step1 = await confirm('Start?');
if (step1) {
  await doStep1();
  const step2 = await confirm('Continue?');
  if (step2) {
    await alert('Done!', 'Success', { type: 'success' });
  }
}
